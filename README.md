# dev-spaces-minimal
Minimum Dev Spaces environment with all the plugins and tools that I like to use to do my work

CLI and runtime tools and configuration:
    - Configures `git` with my account information (change to yours if using)
    - Updates `Node.js`
    - Installs Claude code
    - `istioctl`
    - `kustomize`
    - `helm`
    - `ansible`
    - `oc` is already installed, but may be outdated
    - `.vscode` plugins and settings
